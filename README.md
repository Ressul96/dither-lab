# Dither Lab

Local-first dithering studio built with Tauri, vanilla HTML/CSS/JS, and a graph-driven preview pipeline.

## Current Focus

- Node graph editing and graph-driven preview
- Video source playback, trim, compare modes, and frame stepping
- Dither, adjust, blur, glow, distort, and mix nodes
- Current-frame still export through the export sheet
